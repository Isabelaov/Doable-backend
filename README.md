## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript Doable repository.

## Project setup

```bash
$ yarn install
```

- Remember to fill the environment variables in the file `template.env` into a `.env` file to run the project

The database used is MYSQL

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
